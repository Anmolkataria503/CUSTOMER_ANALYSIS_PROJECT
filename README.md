# Customer Behaviour Analysis Project

## Project Overview

This project analyzes customer shopping behavior data using Python, MySQL, and Power BI. The goal is to identify purchasing patterns, customer segments, revenue drivers, and business insights that can help companies make better marketing and sales decisions.

---

## Business Requirement

The company wants to understand:

* Which customer groups generate the highest revenue
* Which products are purchased most frequently
* How discounts influence customer purchases
* Which products receive the best customer ratings
* The contribution of different age groups to total sales
* The behavior of repeat and loyal customers
* The impact of subscription status on purchasing activity

---

## Tools & Technologies Used

### Python (Pandas)

Python was used for:

* Data cleaning
* Handling missing values
* Removing duplicates
* Data transformation
* Creating new calculated columns (Age Group, Customer Segments, etc.)
* Preparing data for analysis

Why Python?

Python allows efficient processing of large datasets and helps convert raw data into an analysis-ready format.

---

### MySQL

MySQL was used for:

* Storing cleaned customer data
* Writing analytical SQL queries
* Customer segmentation
* Revenue analysis
* Product performance analysis
* Discount impact analysis

Why MySQL?

SQL helps extract business insights from structured data using powerful queries and aggregations.

---

### Power BI

Power BI was used for:

* Interactive dashboards
* Data visualization
* KPI monitoring
* Revenue analysis
* Customer behavior analysis
* Executive-level reporting

Why Power BI?

Power BI converts analytical results into visual dashboards that help business stakeholders make data-driven decisions.
